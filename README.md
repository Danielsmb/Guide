# Gambar CS Line

Website panduan lengkap gambar cara betting togel, deposit, dan informasi bank untuk CS Line.

## 🌐 Demo
[Live Demo](https://namaproject.vercel.app)

## 🚀 Teknologi
- HTML5
- CSS3
- JavaScript (Vanilla)
- Deploy via Vercel

## 📂 Struktur Folder

├── index.html # Halaman utama
├── betting.html # Cara betting togel
├── event.html # Event line togel
├── deposit.html # Cara deposit
├── pltest.html # Betting PL Test
├── bank.html # Profile Bank & E-wallet
├── qris.html # RRN Qris
├── style.css # Styling
├── script.js # JavaScript
├── vercel.json # Konfigurasi Vercel
└── README.md # Dokumentasi

## 🛠️ Development Lokal
1. Clone repository
2. Buka `index.html` di browser atau gunakan Live Server

## 📦 Deploy ke Vercel
1. Push code ke GitHub
2. Import repository di [vercel.com](https://vercel.com)
3. Klik Deploy
